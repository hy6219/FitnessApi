## FitnessApi

해당 서비스는 현재 진행 단계에 있는 "Bridge"서비스 진행 단계 중 고려하게 된
REST API 서비스입니다

### 목적 : Spring data rest (with HATEOAS)를 적용한 피트니스 센터 정보에 대한 REST API 구축

❤ 사용한 기술 및 도구

- SpringBoot
- Spring data rest
- Spring data jpa
- Gradle
- HATEOAS
- Swagger(Swagger Boot Starter)
- jacoco
- JUnit5
- Spring web mvc
- yaml
- InteilliJ
